For lsof, to show opened files for certain path only, put -- in front of each path

lsof -- /home4
lsof -- /home4 -- /home2

lsof will show all opened file containing the path.

lsof is useful to find out all file(s) opened bya particular process.

-i Lists IP sockets. i for everything or sort 4 for ipv4, 6 for ipv6, UDP/TCP, or :range-range for specific ports
-n Do not resolve hostnames (no DNS).
-u List files opened by processes belonging to a specific user
-p display opened files based on PID numbers (exclude with ^PID eg ^2052)
-P Do not resolve port names (list port number instead of its name).
-t IDs of processes that have opened a particular file
+D lsof search for all open instance of diretory and subs
-R list Parent Process ID
  • -p [PID] -R for parent process id of specific process

One can also list Unix Sockets by using lsof -U.

lsof +L1 find files that are unlinked
lsof -i information about processes with open ports
