Openssl can encrypt and decrypt files. Sometimes these have the .enc extension

openssl enc -ciphers list all ciphers available to openssl
openssl <file> -d -salt -md md5 -in <.enc> -out <file> -k <pass> use openssl to decrypt a file
<file> to decrypt
-d: decrypt
-salt: salted
-md: hash
-in: encrypted file
-out: decrypted file output
-k: password to decrypt file with

potentially useful script https://github.com/glv2/bruteforce-salted-openssl
Can provide candidates for passwords from a file. Downside is you still have to loop through algos to try and actually decrypt
