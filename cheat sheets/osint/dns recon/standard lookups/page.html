Standard Lookup

Dig
$ dig <domain>

DNSRecon
$ dnsrecon -t std -d <domain>

Only return IP address
$ dig <domain> +short

Use specific NS
$ dig @<name server> <domain>

Reverse DNS lookup
$ dig -x <ip> +short

Reverse IP lookup for range
$ dnsrecon -t rvs -i 192.168.1.1,192.168.1.254

Query specific type
$ dig -t <type> <domain> <other options>
