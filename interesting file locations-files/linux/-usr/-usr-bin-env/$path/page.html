/usr/bin/env is a variable that we can control, we have access to the variables in /usr/bin/env

$PATH is one such variable that we can typically control

set $PATH=/tmp:$PATH sets the /tmp folder as the first folder to look for executables in the PATH (run our own verion of 'ls' or 'man' for example instead of those in /bin/ or /usr/bin/)


printenv print all or part of environemnt
env print all exported environment or run a program in a modified environment
set print the name and value of each shell variable
